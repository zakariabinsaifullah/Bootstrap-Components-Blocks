== Description ==

<strong>Components Blocks</strong> is a collection of <strong>Custom Gutenberg Blocks</strong> built with <strong>Bootstrap Framework</strong>. All blocks designs are taken from <i><b>Bootstrap Components</b></i>.

**_Right Now, the following blocks are included-_**

- **Bootstrap Container Block**: It allows you to create a container section. You can set either **Boxed Container** or **Fluid Container**. This block includes the following key features:-

  - **_Background Color or Background Image_**
  - **_Background Overlay with Opacity Control_**
  - **_Allow All Inner Blocks_**

- **Bootstrap Button Block**: A complete set of Buttons with different types e.g Primary, Secondary, Info, Success, Danger, etc.

- **Bootstrap Alert Block**: A complete set of Alerts with different types e.g Primary, Secondary, Info, Success, Danger, etc.

- **Bootstrap Progress Bar Block**: A complete set of Alerts with different types e.g Primary, Secondary, Info, Success, Danger, etc. Animated and Striped Bars are also available to set.
